# Security Policy

## Reporting a Vulnerability

We take the security of our AI infrastructure and internal tools seriously.

If you discover a vulnerability in any project owned by **Musabdulai IO**, please do **not** open a public issue.

**Please email findings directly to:**
[hello@musabdulai.com](mailto:hello@musabdulai.com)

We will acknowledge receipt within 48 hours and provide a timeline for remediation.
